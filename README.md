# 🤖 First Choice Private School Website

## 📌 Project Overview
This project is a website created to provide information about **First Choice Private School**.  
The website also promotes learning in **Artificial Intelligence (AI)** and **Robotics**.

It includes modern design elements such as animations, images, and interactive sections.

---

## 🎯 Purpose of the Project
The purpose of this project is to:

- Provide information about First Choice Private School
- Promote learning in Artificial Intelligence and Robotics
- Help parents and students contact the school
- Demonstrate how to build a website using HTML and CSS

---

## 🖥️ Technologies Used
- HTML  
- CSS  
- GitHub (for hosting the website)
